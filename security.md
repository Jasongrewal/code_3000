# Security

## Intended Users
The intended users of this repository are students and instructors in CSE 3000: Contemporary Issues in Computer Science and Engineering. The code and data are primarily used for coursework, assignments, and demonstrations related to computing topics.

## Security Risks
The security risk for this repository is low because it mainly contains coursework and example datasets. However, if the data or code were misused, it could potentially expose assignment solutions or be copied without permission. There is no sensitive personal information or confidential data stored in this repository.

## Security Measures
Some steps have been taken to maintain security for this repository:
- A CODEOWNERS file exists to define repository ownership.
- Pull request rules require code owner review before merging changes.
- The repository does not store sensitive data such as passwords or private information.

Because the repository only contains coursework materials, additional security protections are not necessary.
